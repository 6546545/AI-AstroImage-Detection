# Production Deployment Guide

## Quick Start
```bash
# Deploy the system
python3 deploy.py

# Start production
./start_production.sh
```

## System Requirements
- Python 3.8+
- 5GB+ disk space
- 2GB+ RAM
- Virtual environment recommended

## Configuration
Edit `production_config.yaml` to customize settings.

## Monitoring
Check logs in `logs/` directory.
